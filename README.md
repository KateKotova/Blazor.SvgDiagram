# Реализация тестового задания "Прототип графического редактора на основе Svg.js" (Svg.js Graphic Editore)
![Скриншот](./screenshots/screenshot.jpg)

## Средства разработки
- **Язык программирования**: JavaScript.
- **Библиотека для манипуляции и анимации векторной графики SVG**: svg.js 3.2.
- **Клиентский фреймворк для создания интерактивных веб-приложений**: Blazor WebAssembly (WASM).

## Описание
Я создала прототип графического редактора на svg.js за 1 неделю и потом реализовала MVP графического редактора на svg.js за 3 месяца.

В данном прототипе редактора можно создавать следующие элементы:
- прямоугольники;
- круги;
- линии.

Каждый создаваемый объект имеет изначально произвольную позицию, размеры, заливку и обводку.

Возможности работы с объектами:
- выделение элементов по одному или в группу с помощью клавиши <Ctrl>;
- удаление выделенных элементов.

Также в редакторе можно:
- изменять размеры холста;
- включать или выключать сетку;
- назначать шаг сетки.
